// CommonTasks YAML Reference: SharePointSetup
// ===========================================

:YmlCategory: SharePointSetup

:abstract:    {YmlCategory} installs SharePoint Server binaries, prerequisites, and language packs.

[#dscyml_sharepointsetup]
= DSC Resource '{YmlCategory}'

[[dscyml_sharepointsetup_abstract, {abstract}]]
{abstract}

.When to use '{YmlCategory}'
****
Use this resource when you need to install SharePoint Server on a machine. It handles the prerequisites installer, the main SharePoint binary installation, and any language packs. This must be applied before SharePointProvisioning can create or join the farm.
****
