.When to use '{YmlCategory}'
****
Use this resource when you need to add, modify, or remove entries in the hosts file (C:\Windows\System32\drivers\etc\hosts). This is useful for local name resolution overrides such as pointing hostnames to specific IPs during migrations, testing, or when DNS is not available.
****

| Parameter

| Allowed Values

| HostName
| Key
| String
| Specifies the name of the computer that will be mapped to an IP address.
|

| IPAddress
| 
| String
| Specifies the IP Address that should be mapped to the host name.
|

| Ensure
|
| String
| Specifies if the hosts file entry should be created or deleted.
| - *Present* (default)
  - Absent

|===


.Example
[source, yaml]
----
HostsFileEntries:
  Entries:
    - HostName:  server01
      IPAddress: 192.168.0.1
    - HostName:  server02
      IPAddress: 192.168.0.2
      Ensure:    Present
    - HostName:  server03
      Ensure:    Absent
----
