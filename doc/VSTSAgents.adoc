[#dscyml_vstsagents]
= DSC Resource '{YmlCategory}'

[[dscyml_vstsagents_abstract, {abstract}]]
{abstract}

.When to use '{YmlCategory}'
****
Use this resource when you need to deploy Azure DevOps pipeline agents on machines for running CI/CD build and release pipelines. It handles agent download, installation, configuration with the Azure DevOps organization, and service registration.
****
