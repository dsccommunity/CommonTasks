:YmlCategory: WebVirtualDirectories

:abstract:    {YmlCategory} creates IIS virtual directories that map URL paths to physical directories.

[#dscyml_webvirtualdirectories]
= DSC Resource '{YmlCategory}'

[[dscyml_webvirtualdirectories_abstract, {abstract}]]
{abstract}

.When to use '{YmlCategory}'
****
Use this resource when you need to create virtual directories within an IIS website or web application. Virtual directories map a URL path to a physical folder on disk (or a network share), allowing content to be served from locations outside the website's root directory.
****

.Example
[source, yaml]
----

  Items:
    - Name: VirtualDirectory1
      PhysicalPath: C:\InetPub\VirtualDirectory1
      WebApplication: TestApp1
