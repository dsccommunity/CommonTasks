:YmlCategory: ExchangeAutoMountPoints

:abstract:    {YmlCategory} configures Exchange Server automatic mount points for database and log storage volumes.

[#dscyml_exchangeautomountpoints]
= DSC Resource '{YmlCategory}'

[[dscyml_exchangeautomountpoints_abstract, {abstract}]]
{abstract}

.When to use '{YmlCategory}'
****
Use this resource when you need to set up the disk mount point structure for Exchange Server mailbox database and transaction log volumes. Exchange best practices recommend dedicated volumes for databases and logs, and this resource automates the mount point configuration following those practices.
****

.Attributes of category '{YmlCategory}'
[cols="1,1,1,2a,1a" options="header"]
|===
| Parameter
| Attribute
| DataType
| Description
| Allowed Values

| Identity
|
| String
| The server identity (defaults to node name)
|

| AutoDagDatabasesRootFolderPath
| Mandatory
| String
| The root folder path for Exchange databases
|

| AutoDagVolumesRootFolderPath
| Mandatory
| String
| The root folder path for Exchange volumes
|

| DiskToDBMap
| Mandatory
| String[]
| The disk to database mapping
|

| SpareVolumeCount
| Mandatory
| UInt32
| The number of spare volumes to keep available
|

| EnsureExchangeVolumeMountPointIsLast
|
| Boolean
| Ensure Exchange volume mount point is last
| Default: `$false`

| CreateSubfolders
|
| Boolean
| Whether to create subfolders
| Default: `$false`

| FileSystem
|
| String
| The file system to use
| - *NTFS* (default)
  - REFS

| MinDiskSize
|
| String
| Minimum disk size
|

| PartitioningScheme
|
| String
| The partitioning scheme to use
| - MBR
  - *GPT* (default)

| UnitSize
|
| String
| The allocation unit size
| Default: `64K`

| VolumePrefix
|
| String
| The volume prefix
| Default: `EXVOL`

|===


.Example
[source, yaml]
----
ExchangeAutoMountPoints:
  Identity: \ExchangeServer01
  AutoDagDatabasesRootFolderPath: C:\ExchangeDatabases
  AutoDagVolumesRootFolderPath: C:\ExchangeVolumes
  DiskToDBMap:
    - DAG01-DB1
    - DAG01-DB2
  SpareVolumeCount: 1
  VolumePrefix: EXVOL
----
