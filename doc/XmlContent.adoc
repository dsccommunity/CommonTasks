.When to use '{YmlCategory}'
****
Use this resource when you need to modify XML configuration files on target nodes. It uses XPath expressions to precisely target and modify elements, attributes, or text within XML files. This is useful for application configuration files like app.config, web.config, or any custom XML configuration that needs specific values set.
****

.Example
[source, yaml]
----
XmlContent:
  - Path: D:\web.config
    Ensure: Present
    XPath: /configuration/appSettings/Test1/Test1
    Attributes:
      TestValue1: 'SomeValue1'
      TestValue2: 'SomeValue2'
  - Path: D:\web.config
    Ensure: Present
    XPath: /configuration/appSettings/Test1/Test2
    Attributes:
      TestValue1: 'SomeValue1'
      TestValue2: 'SomeValue2'
----
