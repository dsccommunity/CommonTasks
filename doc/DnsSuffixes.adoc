:YmlCategory: DnsSuffixes

:abstract:    {YmlCategory} configures DNS connection-specific suffixes on network interfaces.

[#dscyml_dnssuffixes, {YmlCategory}]
= DSC Resource '{YmlCategory}'

[[dscyml_dnssuffixes_abstract, {abstract}]]
{abstract}

.When to use '{YmlCategory}'
****
Use this resource when you need to configure DNS suffixes on individual network adapters. DNS suffixes enable short-name resolution by appending domain names to unqualified queries, which is important for proper name resolution in multi-domain environments.
****

|===
| Parameter
| Attribute
| DataType
| Description
| Allowed Values

| InterfaceAlias
| Mandatory
| String
| Alias of the network interface for which the DNS suffix is set.
|

| ConnectionSpecificSuffix
| Mandatory
| String
| DNS connection-specific suffix to assign to the network interface.
|

| Ensure
|
| String
| Specifies whether the DNS suffix should exist.
| - Absent
  - Present

|===


.Example
[source, yaml]
----

  Suffixes:
    - InterfaceAlias: en0
      ConnectionSpecificSuffix: contoso.com
----
