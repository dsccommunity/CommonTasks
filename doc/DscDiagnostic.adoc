.When to use '{YmlCategory}'
****
Use this resource when you need to set up restricted, secure JEA endpoints for managing DSC operations remotely. It creates role capabilities and session configurations that allow operators to perform DSC diagnostics (triggering auto-correct, refresh, or monitoring) without requiring full administrative access.
****
