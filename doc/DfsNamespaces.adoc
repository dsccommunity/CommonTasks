[#dscyml_dfsnamespaces, {YmlCategory}]
= DSC Resource '{YmlCategory}'

[[dscyml_dfsnamespaces_abstract, {abstract}]]
{abstract}

.When to use '{YmlCategory}'
****
Use this resource when you need to set up a DFS namespace for unified file share access across your organization. DFS namespaces provide a virtual view of shared folders from multiple servers under a single path, simplifying user access and enabling transparent failover between file servers.
****

.Attributes of category '{YmlCategory}'
[cols="1,1,1,2a,1a" options="header"]
|===
| Parameter
| Attribute
| DataType
| Description
| Allowed Values

| DomainFQDN
| Mandatory
| String
| The fully qualified domain name (FQDN) of the domain.
|

| NamespaceConfig
| Mandatory
| Hashtable[]
| List of DFS namespace configurations to manage
|

|===


.Example
[source, yaml]
----
DfsNamespaces:
  DomainFqdn: contoso.com
  NamespaceConfig:
    - Sharename: AdminShare
      Targets:
        - DscFile01
        - DscFile02
    - Sharename: Software
      Targets:
        - DscFile01
        - DscFile02
----
