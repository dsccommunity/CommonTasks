= DSC Resource '{YmlCategory}'

[[dscyml_securitybase_abstract, {abstract}]]
{abstract}

.When to use '{YmlCategory}'
****
Use this resource when you need to apply a standardized security baseline to your servers. It configures security options, user rights assignments, and account policies based on whether the server is a member server or domain controller. This provides CIS/STIG-aligned security hardening appropriate to the server role.
****
