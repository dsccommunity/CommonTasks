= DSC Resource '{YmlCategory}'

[[dscyml_network_abstract, {abstract}]]
{abstract}

.When to use '{YmlCategory}'
****
Use this resource when you need to configure network-level settings on interfaces categorized by network zone. It manages MTU size, protocol bindings, DNS server addresses, WINS settings, and NetBIOS configuration. For IP address assignment and routing, use IpConfiguration or NetworkIpConfiguration instead.
****
