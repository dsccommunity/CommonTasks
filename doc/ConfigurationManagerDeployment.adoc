:YmlCategory: ConfigurationManagerDeployment

:abstract:    {YmlCategory} deploys and installs Microsoft Endpoint Configuration Manager (SCCM/MECM) infrastructure including prerequisites, ADK, MDT, SQL, and the CM site server.

[#dscyml_configurationmanagerdeployment]
= DSC Resource '{YmlCategory}'

[[dscyml_configurationmanagerdeployment_abstract, {abstract}]]
{abstract}

.When to use '{YmlCategory}'
****
Use this resource when you need to perform a fresh installation of a Configuration Manager site server. It handles prerequisite Windows features, installs the Windows Assessment and Deployment Kit (ADK), Microsoft Deployment Toolkit (MDT), SQL Server, and the Configuration Manager setup itself. This is typically used for greenfield SCCM deployments.
****
